# Campaign Intelligence Dashboard

A marketing campaign calendar dashboard integrating MoEngage + Google Analytics 4.

## Features
- Campaign calendar with month/week view
- Color-coded channels (Push, Email, WhatsApp, SMS)
- Performance metrics per campaign
- Collision detection for overlapping campaigns
- Charts with Recharts

## Tech Stack: Next.js 14, TailwindCSS, FullCalendar, Recharts, Vercel

## Setup
```bash
npm install && npm run dev
```

Set environment variables in .env.local (see .env.example)
